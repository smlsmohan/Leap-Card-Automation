The report elaborates on the design, development, and deployment of the RPA solution "LeapCard_Automation," which aims to automate the validation of student Leap Card applications. The project purposes to solve the inefficiencies and scope for human-induced errors in manually processing these applications.


Currently, the manual system can be described as repeating tasks, very slow processing, and inconsistent application of business rules. This automation wants to develop a strong and reliable process-facing high volumes of applications with speed and accuracy.

The project scope includes the following:

--> Read applicant data from a structured file system

--> Apply a series of business rules to each application for validation

--> Make decisions to approve, put under manual review, or reject applicationsGenerate consolidated output reports in CSV format

--> Automatically inform applicants of their approval status via email

--> The solution is developed in UiPath Studio and is meant to execute on a local machine. All file operations use relative paths to maintain portability and provide offline use.
